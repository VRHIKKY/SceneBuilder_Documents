# アップデート項目
## 機能調整
### 操作性向上
* アセットパネル、設置アイテム一覧でマウスホイールを回転させたときのスクロール量を3倍程度に増やしました。
* シーンパネルで、マウスのホイールを回転させたときのカメラ移動量を3倍程度に増やしました。

### アセット配置時の動作改善
* Imageを設置したとき、アスペクト比が強制的に1:1になっていたので、元の画像のアスペクト比を維持するようになりました。

## 不具合修正
* テキスト入力ボックスの位置がずれることがある問題を修正しました。
* Scriptをダブルクリック（連続してクリック）して、スクリプトエディタを開くと、保存や別のスクリプトを開くことができなくなる問題を修正しました。
* アセットを多数配置し、設置アイテム一覧がスクロールできるとき、ファイルメニューを開くことや、アセットパネルの操作ができなくなることがある問題を修正しました。
